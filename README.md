This data analysis is based on an open access dataset published by sharkattackfile.net and shared by Kaggle. 

The file involves over 6,000 incidents of shark attacks on humans.

During my analysis I tried to point:
- countries and areas with the most recorded attacks,
- mediane length of a shark attacking humans,
- gender, age, activity doing during the attack and cause of the attack (provoked/unprovoked) for the most common group of victims, 
- most frequent injuries,
- hours and months with the highest number of cases. 

The analysis was made as a part of data analysis course by Kajo Rudziński, kajodata.com.
